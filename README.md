# Base Utils Toolkit

Small helper contracts and notes for working with the **Base** network.

- **Author:** [Alexpro780](https://github.com/Alexpro780)
- **Networks:** Base Mainnet (8453), Base Sepolia (84532)
- **Purpose:** Provide simple, reusable building blocks and constants for other Base projects.

## What is inside?

### `contracts/BaseNetworkConfig.sol`

A small contract that exposes common chain IDs and placeholder RPC URLs for Base networks.

### `contracts/BaseAddressBook.sol`

A tiny helper that stores important addresses (for example USDC, routers, dApps) for Base Mainnet.
You can adjust the addresses later if something changes.

## How to use

1. Upload this folder to a new public GitHub repository, for example `base-utils-toolkit`.
2. Later, when you build other Base projects, you can import these contracts or simply copy‑paste the parts you need.
3. The main goal is to show real, open‑source, Base‑related code in your public profile.

Even if ты пока не программист, наличие такого репозитория делает твой профиль более «техническим» для аналитических систем.
